Welcome to Sociagraph!

Sociagraph is a web application equipped with tools for Natural Language Processing (NLP).

These are the current tools available:

  • Key Information Extractor. This tool extracts key information related to the themes provided.
  • Sentiment Analyzer. This tool analyze the sentiment of each sentence in a given text. Each sentence will be labeled happy, sad, angry, fearful or neutral.
